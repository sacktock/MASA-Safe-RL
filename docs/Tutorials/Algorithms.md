# Algorithms

These tutorials show how to configure and compare MASA algorithms in small, readable experiments.

For a verified list of the algorithms currently present in the codebase, start with [Algorithms Overview](../Algorithms/Overview).

- [Tabular Safe RL Baselines](Algorithms/Tabular%20Safe%20RL%20Baselines) compares `q_learning`, `q_learning_lambda`, `lcrl`, `sem`, and `recreg` on one constrained tabular environment.
- [Continuous Safe RL Baselines](Algorithms/Continuous%20Safe%20RL%20Baselines) is a stub for PPO, CPO, and PPO Lagrangian continuous-control baselines.
